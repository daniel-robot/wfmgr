# Handoff Ready Checklist

A handoff is ready when:

- [ ] The current goal is clear.
- [ ] The current state is summarized.
- [ ] The next agent is identified.
- [ ] Relevant files or folders are listed.
- [ ] Constraints are listed.
- [ ] What to do is clear.
- [ ] What not to do is clear.
- [ ] Expected output is clear.
- [ ] Validation commands are included if relevant.
- [ ] Risks or watchouts are listed.
