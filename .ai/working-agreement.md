# AI Agile Team Working Agreement

1. Make small, incremental changes.
2. Do not edit files without first stating the intended files.
3. Do not overwrite another agent's work.
4. Before editing, check .ai/locks/.
5. After editing, update .ai/handoffs/current-handoff.md.
6. Prefer tests or validation commands with every implementation change.
7. Keep architecture decisions in .ai/decisions/.
